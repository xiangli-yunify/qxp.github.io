青云全象云平台面向企业数字化的软件需求提供平台和应用服务，是一个包含快速应用开发能力、含有一定数量标品、能与存量系统集成、能联通各业务系统、以及能处理跨系统数据管理的Paas 平台。例如：生产制造 ERP、采购和供应链、营销 CRM、进销存物管、客户服务、财务、行政管理 OA、人事管理、费控、在线沟通、协同办公（包括项目管理、文档管理等）。

功能特性

青云全象云低代码平台可提供的功能有：

  • 可视化建模
  • 灵活的页面引擎
  • 移动端支持
  • 面向开发者友好
  • 集成到 QXP PaaS 平台

可视化建模

可视化建模包含灵活的表单引擎、可视化流程引擎、应用间数据互通、外部接口代理、用户自定义任务、复杂功能规则引擎等内容。

灵活的表单引擎

动态定义数据模型，支持丰富的数据项格式。灵活定义数据详情页面，支持关联数据的自定义展示方式，高性能展示批量关联数据。

可视化流程引擎

可视化定义流程，用于审批流程、工单协作等场合的行为编排。支持触发自定义任务、动态节点管理。

应用间数据互通

每个数据模型都可以一键对外开放 API，多个应用之间业务数据可以互通关联。自定义的复杂规则也可以对外开放接口。

外部接口代理

外部 API 和 RPC 接口可以被抽象为服务代理访问，无需代码即可对第三方接口进行访问和包装。

用户自定义任务

可以定义定时任务、即时执行任务，以实现数据的批量处理，或 RPA 操作。

复杂功能规则引擎

在视图中定义一个复杂的业务流程，包括逻辑控制和对本地、第三方的调用，数据表处理等。可以被流程和定时任务、即时任务调用。

灵活的页面引擎

灵活的页面引擎支持自定义页面布局、丰富的系统控件、自定义控件扩展、丰富的布局模板等内容。

自定义页面布局

可以灵活自定义页面的布局、定义页面的各种元属性；可以自定义各级组件的属性、样式、事件等，实现页面的可视化编辑。

丰富的系统控件

内置常用的标准页面 html 控件、表单类控件、多媒体控件、富文本控件等，满足通常页面开发所需。

自定义控件扩展

若业务使用到的控件系统不支持，或想要抽取出业务自己的控件，可以按平台定义的扩展格式编写控件。

丰富的布局模板

包含管理类页面、站点类页面、报表类页面模板，满足快速开发一个灵活、美观页面的需求。

移动端支持

  • 支持将应用发布为 iOS/android 原生应用。
  • 支持应用发布为微信、钉钉等小程序。
  • 支持应用布局排版为 h5 页面，在手机端可以共享和操作。
  • 支持发布到青云 workly.ai 的亿洽即时聊天平台第三方接口，实现跨域消息推送。

面向开发者友好

面向开发者友好包含面向二次开发封装、插件控件版本管理、开发者文档、开发和生产环境同步等内容。

面向二次开发封装

支持 groovy 脚本定义业务逻辑，支持页面自定义控件，独立服务可以通过平台微服务化部署。

插件、控件版本管理

插件和控件引入版本管理，系统升级时不会直接影响线上版本体验，维护人员可以手动更新到最新版本。

开发者文档

通过丰富的文字、图片、视频等方式以及页面中的引导，减少开发者使用的困惑。

开发、生产环境同步

应用创建后区分开发环境和生产环境，应用维护可以充分测试后一键同步到生产环境，实现安全切换。
